Con-Troll Convention Management integration plugin

== Description ==

This plugin implement integration functions for a Wordpress powered convention web site to work with
the Con-Troll convention management system.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/con-troll` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Settings->ConTroll screen to configure the plugin
